# 🤖 AI Impact on Students – Data Science Project

## 📌 About the Project

This is my first Data Science project built using Python. The project explores the impact of Artificial Intelligence (AI) on students and analyzes how AI tools influence academic performance, study habits, dependency levels, burnout risk, and skill retention.

I started working on this project on **1 June 2026** and successfully completed it on **20 July 2026**. This project marks the beginning of my journey in Data Science and helped me gain hands-on experience in data analysis, visualization, and building interactive dashboards.

---

## 🎯 Objectives

* Analyze the adoption of AI among students.
* Study the relationship between AI usage and academic performance.
* Examine AI dependency and burnout levels.
* Evaluate prompt engineering skills and skill retention.
* Generate meaningful insights through data visualization.
* Build an interactive web application using Streamlit.

## 📊 Dataset Information

The dataset contains information about students, including:

* Student ID
* Major Category
* Year of Study
* Pre-Semester GPA
* Weekly GenAI Hours
* Primary Use Case of AI
* Prompt Engineering Skill
* Tool Diversity
* Paid Subscription Status
* Traditional Study Hours
* Perceived AI Dependency
* Institutional Policy
* Anxiety Level During Exams
* Post-Semester GPA
* Skill Retention Score
* Burnout Risk Level

## 💡 Key Insights

* Students using AI tools extensively showed noticeable changes in academic performance.
* Prompt engineering skills play an important role in effectively utilizing AI tools.
* AI dependency and burnout are emerging factors among students.
* Skill retention varies based on study habits and AI usage patterns.
* Generative AI is transforming the way students learn and complete academic tasks.
